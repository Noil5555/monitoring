Just a little projekt. A Monitorung system with runs in the background.
Build for a Linux System, but works just fine on windows as well.
Once started this script runs every 5 minutes.
If some of the heardware reatches the critical value or chages its status, the skript will sent a e-mail to the specivied e-mail account.
The Mail account of you sender and reciver is specified in the config.json.
There you have so add your sender, the monitoring system and the mail you want to get the notifications.
